Course Description:
This course provides a rigorous introduction to the theory and practice of machine learning. Students will develop a strong mathematical foundation covering linear algebra, probability, and optimization, and apply that foundation to implement and evaluate machine learning systems. Topics span supervised learning (regression, classification, ensemble methods), unsupervised learning (clustering, dimensionality reduction), and an introduction to deep learning (feedforward networks, CNNs, RNNs, and Transformers). A central theme is the scientific methodology of ML: problem framing, dataset construction, model selection, evaluation, and reproducibility.

Prerequisites: CS201 (Data Structures), MATH301 (Linear Algebra), MATH302 (Probability and Statistics). Students without the math prerequisites should speak with the instructor before enrolling.

Learning Objectives:
By the end of this course, students will be able to:
1. Explain the mathematical foundations of common ML algorithms including gradient descent, maximum likelihood estimation, and the bias-variance tradeoff.
2. Implement supervised learning algorithms from scratch in Python/NumPy and evaluate them rigorously using cross-validation and held-out test sets.
3. Train and fine-tune neural networks using PyTorch, including CNNs for image tasks and sequence models for text.
4. Apply unsupervised learning methods including k-means clustering and PCA to real datasets.
5. Critically evaluate ML papers, identifying methodological strengths and weaknesses.
6. Build a complete end-to-end ML pipeline for an applied problem of their choosing (capstone project).
7. Articulate ethical considerations in ML system design including fairness, privacy, and accountability.

Course Schedule (abbreviated):

Week 1–2: Foundations
  - The ML pipeline: problem framing, data, model, evaluation
  - Supervised learning setup: loss functions, empirical risk minimization
  - Linear regression: closed-form solution, gradient descent, regularization (Ridge, Lasso)
  - Reading: Murphy Ch. 1–3

Week 3–4: Classification
  - Logistic regression: decision boundaries, cross-entropy loss, maximum likelihood
  - Support Vector Machines: margin maximization, kernel trick
  - Decision trees and random forests: information gain, bagging, feature importance
  - Reading: Murphy Ch. 4–5, Breiman (2001) on random forests

Week 5–6: Evaluation and Methodology
  - Cross-validation, train/val/test splits, data leakage
  - Metrics: accuracy, precision/recall, F1, AUC-ROC, calibration
  - Hyperparameter tuning: grid search, random search, Bayesian optimization
  - Bias-variance decomposition in depth

Week 7: Midterm

Week 8–9: Unsupervised Learning
  - K-means and hierarchical clustering: algorithms, initialization, evaluation
  - Gaussian Mixture Models and EM algorithm
  - PCA: eigendecomposition, explained variance, applications
  - Autoencoders as nonlinear dimensionality reduction

Week 10–11: Deep Learning
  - Feedforward networks: backpropagation derivation, activation functions
  - CNNs: convolution operation, pooling, receptive fields, modern architectures
  - RNNs and LSTMs: sequence modeling, vanishing gradients
  - Transformers and attention: self-attention mechanism, positional encoding
  - Transfer learning and fine-tuning

Week 12: Advanced Topics and Capstone Presentations
  - Reinforcement learning overview
  - ML ethics and societal impact
  - Capstone project presentations (final exam period)

Required Texts:
- Murphy, K.P. (2022). Probabilistic Machine Learning: An Introduction. MIT Press. (Free PDF online)
- Goodfellow, I., Bengio, Y., Courville, A. (2016). Deep Learning. MIT Press. (Free PDF online)

Recommended: Bishop, C. (2006). Pattern Recognition and Machine Learning. Springer.
